# Hardware Prerequisites

| Item | Notes |
|------|-------|
| Raspberry Pi Zero 2W | Must have 40-pin header soldered (WH variant) |
| Raspberry Pi AI Camera | Includes 15-pin cable — **not** suitable for Zero 2W |
| 22-pin to 15-pin CSI adapter cable | Required for Zero 2W camera connection |
| Inky Impression 13.3" | 2025 Edition recommended for shorter refresh |
| MicroSD card (≥16 GB, class 10) | Raspberry Pi OS Lite (64-bit) |
| Micro USB power supply (≥2.5A) | Pi Zero 2W under load draws ~1.5A |
| Background image file | Van Gogh café at 1600×1200, pre-prepared |
